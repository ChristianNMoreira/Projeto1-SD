# Projeto 1 - Unidade Lógica Aritmética

Projeto de uma Unidade Lógica Aritmética de 4 bits e 8 operações.

## Operações

- [x] Soma
- [x] Subtração em complemento de 2
- [x] Incremento +1
- [x] Troca de sinal
- [x] Portas lógicas bit a bit

## Entradas

| K₂K₁K₀ | Operação       |
| ------ | -------------- |
| 000    | Soma           |
| 001    | Incremento +1  |
| 010    | Subtração      |
| 011    | Troca de sinal |
| 100    | OR bit a bit   |
| 101    | XOR bit a bit  |
| 110    | AND bit a bit  |
| 111    | NAND bit a bit |
